# 🧮 BadCalculator — Графічний калькулятор дефектної логіки

`BadCalculator` — це навчальний десктопний застосунок на мові Python із графічним інтерфейсом Tkinter, створений у межах вивчення дисципліни "Інженерія якості програмного забезпечення". Особливістю програми є спеціально допущені дефекти в інтерфейсі та бізнес-логіці для відпрацювання навичок тестування.

---

## ✨ Особливості та можливості

* 🖥️ Простий графічний інтерфейс користувача на базі вбудованого модуля `Tkinter`.
* 🛡️ **Обробка винятків (Фікс багу):** Реалізовано перехоплення помилки `ZeroDivisionError`. При спробі ділення на нуль програма більше не ламається, а виводить на екран безпечне інформаційне повідомлення `Zero Dev`.
* ⚠️ *Специфічна дефектна логіка:* Калькулятор навмисно модифікує фінальний результат будь-якого успішного обчислення (віднімає одиницю).

---

## 🛠️ Використані технології

* **Мова програмування:** Python 3.10+
* **Графічна бібліотека:** Tkinter
* **Тестові фреймворки:** `unittest` (стандартний модуль), `pytest` (автоматизоване модульне тестування)

---

## 🚀 Встановлення та запуск

1. Клонуйте репозиторій проєкту у свою локальну папку:

```bash
git clone https://github.com/anton570/software-quality-engineering.git
cd software-quality-engineering
```

2. Запустіть основний застосунок калькулятора:

```bash
python calculator_bad.py
```

## 🧪 Запуск тестів

Для перевірки працездатності логіки та фіксів у проєкті реалізовано автоматичні тести:

* **Запуск тестів через assert:** `python test_assert.py`
* **Запуск тестів через unittest:** `python -m unittest test_calculator.py`
* **Запуск тестів через pytest:** `pytest test_pytest.py`
